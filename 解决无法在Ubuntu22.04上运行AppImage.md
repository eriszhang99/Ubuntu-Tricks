## 1. 首先下载libfuse
```
sudo apt install libfuse2
```

## 2. 打开相应AppImage文件的权限
右键AppImage 文件并选择属性；转到权限选项卡并选中“允许将文件作为程序执行”。
![image](https://github.com/iamstarlee/Ubuntu-Tricks/assets/44799727/10161011-3206-47c7-bebd-190b6022edf3)

## 3. 输入指令即可运行AppImage文件
```
sudo ./xxx.AppImage
```
